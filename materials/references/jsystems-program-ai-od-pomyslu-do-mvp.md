## Program skeleton (client baseline)

### Day 1 — Fundamenty i misja
- Rewolucja AI, mity, korzyści/ryzyka
- Kluczowe koncepcje: modele, prompt engineering, kontekst
- Mission briefing i setup środowiska
- Research + wymagania projektu

### Day 2 — Od pomysłu do projektu
- Research + planowanie architektury
- UX/UI prototyping
- Przejście z designu do kodu

### Day 3 — Budowa rdzenia aplikacji
- IDE + AI workflows
- Full-stack generation + CLI power
- Debugging i refactoring z AI

### Day 4 — Jakość, bezpieczeństwo, wdrożenie
- Testy (unit/integration/E2E)
- Code review + security scanning
- CI/CD + automatyzacja

### Day 5 — Mistrzostwo, utrzymanie, przyszłość
- Legacy code
- Modele lokalne, prywatność, koszty
- Agentic workflows + MCP
- Podsumowanie i dalszy rozwój
