# Day 3 Prompt Pack

Szybki zestaw promptów do użycia na Day 3.
Źródło pełne: `prompts/02-module-prompts.md`.

## D3-M1 — Mały slice implementacyjny
```text
Zaimplementuj mały slice: <nazwa_slice>.
Zakres: tylko <pliki/moduły>.
Wynik: diff + krótki opis + lista testów do uruchomienia.
Nie zmieniaj nic poza zakresem.
```

## D3-M2 — Debugging
```text
Pomóż zdebugować błąd.
Objaw: <objaw>
Expected: <oczekiwane>
Actual: <faktyczne>
Logi: <fragment>

```

## D3-M3 — Refaktoryzacja
```text
Zrefaktoryzuj <moduł> dla czytelności i utrzymania.

```
